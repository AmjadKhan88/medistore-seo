# Medicine Store SEO Website

Production-ready Next.js 15 marketing and SEO website for an existing React, Node.js, Express, and MongoDB medicine store management system.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Set the variables from `.env.example` in Vercel. The app CTA destination is controlled by `NEXT_PUBLIC_APP_URL`.

## Included

- App Router, TypeScript, Tailwind CSS, strict metadata, sitemap, robots, RSS, OG image, JSON-LD, hreflang and canonical support.
- Marketing pages, blog, documentation, pricing, contact, legal pages, search page, 404 page.
- Analytics hooks for GA4, GTM, Microsoft Clarity, Facebook Pixel, and search engine verification.
- Security headers and CSP in `next.config.ts`.
